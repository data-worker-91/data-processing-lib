# data-processing-lib

A set of low-level Python utilities for tensor data processing and hardware buffer synchronization. 
Designed for research and internal testing of bias-adjustment algorithms in distributed environments.

## Modules:
- `buffer_io_ext.py`: Hardware state synchronization.
- `sys_runtime_init.py`: Environment and logging setup.
- `tensor_util_v4.py`: Data formatting and checksum validation.
